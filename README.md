DESIGN AND INTENT:
Up until this point I always kept my notes on my Terminal! No kidding! I just edited my .zshrc on Linux to echo out strings so that everytime I opened a terminal I would be reminded of things I need to do. My main problem with NoteTaking apps is that they feel too overwhelming. A user that wants to start using note taking apps probably experiences a big influx of different thoughts, which makes them want to have a place where they can organize said thoughts. Many NoteTaking apps seem to jump at the user with too much functuonality and with a ui that makes the user not know where to look.
My philosophy is that if a user wants to be able to do something, a developer should account for the most simplest way to incorporate these features. It should be as easy as a user dragging and clicking their mouse around and not filling up the form every time. I tried to make my app intuitive to use, simple and have a pleasant but minimalistic design. It should be as simple as: I want to go in, take a note, maybe change it if I need and leave.
